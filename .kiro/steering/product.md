# Product Overview

Esports Management UI is a comprehensive back-office system for managing esports teams, players, fixtures, and match results. The application focuses on Call of Duty League (CDL) esports with support for multiple game modes (Hardpoint, Search and Destroy, Overload).

## Core Features

- Team management with game mode win percentages and roster tracking
- Player statistics across different game modes (K/D ratios, objective stats)
- Fixture scheduling and match management
- Results recording with detailed map-by-map statistics
- Client onboarding and API key management
- Match simulation capabilities

## API Integration

The application integrates with a backend API at `https://localhost:7173/api` for all data operations. The system is designed to work with the sitechesports.com API ecosystem.

## Target Users

Back-office administrators and esports organization managers who need to track team performance, schedule matches, and analyze player statistics.
